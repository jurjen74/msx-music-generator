MSX MML Music Generator

Generate MSX chiptune with Claude, preview it in the browser, and export a .vgm for the PSG or MSX-Music chip — playable on MSXgl.

.vgm → plays directly on MSXgl's vgm_player, no Windows tool needed. The Target chip selector decides the format: PSG (AY-3-8910 square waves) or MSX-Music (YM2413 FM — pick the FM voices above). .mml → for the NDP editor route. In-browser playback emulates the PSG only; FM is auditioned via Preview (MML) or on real MSXgl. See msxgl-example/ to build a test ROM. Download .lvgm (compact MSX-optimized VGM, ~75–85% smaller) appears only when the server is started with MSXZIP configured — see the README.
